# install-tests

E2E test runner for LevitateOS installation process.

## Overview

Automated testing infrastructure that boots LevitateOS in QEMU and verifies installation steps work correctly.

## Building

```bash
cargo build
```

## Running Tests

```bash
cargo run
```

## Development

```bash
# Build
cargo build

# Run tests
cargo test

# Check with clippy
cargo clippy
```
